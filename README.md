# Portfolio

> A modern React + Vite portfolio website.

## Stack
- **React** (18)
- **Vite** (6)
- **TypeScript**
- **Tailwind CSS**
- **PNPM** for package management

## Local Setup
```bash
# Install PNPM if you don't have it
npm i -g pnpm

# Install dependencies
pnpm install

# Run development server
pnpm dev
```
The app will be available at `http://localhost:5173`.

## Build
```bash
pnpm build
```
The production build is output to the `dist/` folder.

- **Build workflow**: `.github/workflows/build.yml` – runs on every push, validates the code, and uploads the build artifact.
- **Deploy workflow**: `.github/workflows/deploy.yml` – triggers after a successful build on the `main` branch and publishes to GitHub Pages.
- **Rollback**: If deployment fails, the previous successful `gh-pages` deployment remains live. You can manually revert by re‑deploying a prior commit via the workflow.
